# Cheatsheet Maker - Product Requirements Document

## 产品概述
Cheatsheet Maker 是一个AI驱动的工具，可以将PDF文档或网页链接转换为美观的要点总结卡片（Cheatsheet），支持多语言翻译，并提供响应式预览和PNG导出功能。

## 核心功能

### 1. 内容输入
- **PDF上传**: 支持拖拽或点击上传PDF文件
- **链接输入**: 支持输入网页链接提取内容
- **文件验证**: 验证PDF格式和链接有效性

### 2. AI内容处理
- **内容提取**: 从PDF或网页中提取关键信息
- **要点生成**: 将内容转换为结构化的要点格式
- **翻译功能**: 支持中英文互译，用户可选择最终语言
- **AI模型**: 使用 SiliconFlow 的 moonshotai/Kimi-K2-Thinking 模型

### 3. Cheatsheet生成
- **卡片设计**: 每个要点做成独立卡片
- **现代设计**: 使用Tailwind CSS，符合现代审美
- **色彩丰富**: 使用渐变色彩和现代化配色方案
- **响应式布局**: 兼容桌面端和移动端
- **尺寸限制**: 长宽比例在3:4至2:1之间，适合A4纸打印

### 4. 预览和导出
- **实时预览**: 提供HTML页面预览功能
- **格式选择**: 
  - 桌面端查看（横屏图）
  - 移动端分享（竖屏图）
- **PNG导出**: 高质量PNG图片导出
- **一键下载**: 支持直接下载生成的图片

## 技术架构

### 前端技术栈
- **框架**: React + TypeScript
- **构建工具**: Vite
- **样式**: Tailwind CSS
- **状态管理**: Zustand
- **路由**: React Router
- **UI组件**: 自定义组件 + Lucide图标

### AI服务集成
- **API提供商**: SiliconFlow
- **模型**: moonshotai/Kimi-K2-Thinking
- **认证**: API Token认证
- **环境变量**: API_URL 和 API_TOKEN

### 部署方案
- **代码托管**: GitHub仓库
- **CI/CD**: GitHub Actions自动部署
- **托管平台**: Vercel
- **域名**: 自定义域名支持

## 用户体验要求

### 界面设计
- **现代化**: 简洁、直观的用户界面
- **响应式**: 适配各种屏幕尺寸
- **加载状态**: 提供清晰的加载和进度反馈
- **错误处理**: 友好的错误提示和重试机制

### 性能要求
- **快速响应**: AI请求响应时间优化
- **文件处理**: 支持大文件上传和处理
- **图片导出**: 高质量且快速的PNG生成
- **缓存策略**: 合理的缓存机制提升体验

### 可访问性
- **键盘导航**: 支持键盘操作
- **屏幕阅读器**: 适配辅助技术
- **色彩对比**: 符合WCAG标准
- **多语言**: 中英文界面支持

## 数据安全和隐私

### 文件处理
- **本地处理**: 优先在客户端处理文件
- **数据清理**: 处理完成后及时清理临时数据
- **隐私保护**: 不存储用户上传的敏感内容

### API安全
- **密钥管理**: API密钥安全存储
- **请求限制**: 合理的API调用频率限制
- **错误处理**: 不暴露敏感的错误信息
